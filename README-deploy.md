# Wefaq - Ready Server Version

This package contains the server setup and deployment instructions.

## How to Use

1. Add your React/Vite project files here.
2. Create `.env.local` with:
   GEMINI_API_KEY=your_key_here
3. Install:
   npm install
4. Build:
   npm run build
5. Run:
   npm start

Deploy easily on Render, Vercel, or Netlify.
